# calculator-test

A polished, modern calculator app you can run directly from VS Code.

## Run in VS Code

1. Open the project folder in VS Code.
2. Open the Command Palette and run "Tasks: Run Task".
3. Choose "Run Calculator".
4. Open your browser at http://127.0.0.1:8000/.
5. Click the buttons to perform calculations.

## Run from the terminal

1. Install Node.js if needed.
2. In the project folder, run `npm install`.
3. Start the app with `npm start`.
4. Open http://127.0.0.1:8000/ in your browser.

## Optional desktop app

If you want the Electron desktop wrapper, run `npm run start:electron` after installing the dependencies.

## Files

- `index.html` — calculator page
- `styles.css` — styling
- `script.js` — calculator logic
- `.vscode/tasks.json` — one-click VS Code task
