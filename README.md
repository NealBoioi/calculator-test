# calculator-test

A polished, modern calculator app you can run directly from VS Code.

## Run as a desktop app

1. Open the project folder in VS Code.
2. Start the app with `python3 calculator.py`.
3. A native calculator window will open on your desktop.

## Run from VS Code

1. Open the Command Palette.
2. Run "Tasks: Run Task".
3. Choose "Run Calculator".
4. The app will launch as a desktop window.

## Files

- `index.html` — calculator page
- `styles.css` — styling
- `script.js` — calculator logic
- `.vscode/tasks.json` — one-click VS Code task
